Large text (28px)

With line-height-step: 20px

Line heights rounded up to multiples of 20px:

Small text (12px)

Medium text (20px)

Large text (28px)

Back to normal (16px)

Same Step, Different Font Sizes

All use line-height-step: 30px:

Small text (12px, line-height 1.2 = 14.4px → 30px)

Medium text (16px, line-height 1.4 = 22.4px → 30px)

Large text (24px, line-height 1.6 = 38.4px → 60px)
