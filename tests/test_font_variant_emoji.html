What is font-variant-emoji?

The font-variant-emoji property controls how emoji characters are rendered:

  • normal - Default rendering
  • text - Force text-style rendering (monochrome)
  • emoji - Force emoji-style rendering (colorful)
  • unicode - Use Unicode variation selectors
