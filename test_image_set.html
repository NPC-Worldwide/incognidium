CSS image-set() Function

Responsive Images with image-set()

The image-set() function allows specifying multiple image sources at different resolutions:

background-image: image-set(
    url("image-1x.png") 1x,
    url("image-2x.png") 2x,
    url("image-3x.png") 3x
);

Use Cases

  • 1x - Standard resolution displays
  • 2x - High-DPI/Retina displays
  • 3x - Super high-DPI displays
  • Can also use dpi units (e.g., 300dpi)

Note: The image-set() function is part of CSS Images Module Level 4. It enables browsers to automatically select the most appropriate image resolution based on the device's display density.

Using image-set() ensures optimal image quality while minimizing bandwidth usage, as browsers will download only the resolution they need.
