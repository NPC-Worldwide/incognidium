::backdrop Pseudo-element Demo

What is ::backdrop?

The ::backdrop pseudo-element represents the backdrop behind modal elements like <dialog> and elements with the popover attribute.

Supported Properties

  • background-color - Sets the backdrop background
  • opacity - Controls backdrop opacity
  • backdrop-filter - Applies blur/ effects (when supported)
Note: The ::backdrop pseudo-element is styled through CSS but is only visible when a dialog or popover is actually open/active. In static rendering, the styles are parsed and stored for when they're needed.

The ::backdrop pseudo-element is part of CSS Pseudo-Elements Level 4 and is essential for creating modal overlays with custom styling.
