::selection Pseudo-element Demo

This text uses the default ::selection styling with a coral red background and white text. Try to imagine this text being selected!

This paragraph has custom ::selection styling with a teal background (#4ecdc4) and dark text. Different elements can have different selection colors!

Selection styling helps users understand what content is selected. It's especially useful for:

  • Matching brand colors
  • Improving accessibility
  • Creating cohesive designs

Inline code elements can have their own selection style with a yellow background.

The ::selection pseudo-element only works with a limited set of properties: background-color, color, and text-shadow.
